## ✨ 项目简介

**梦境星云档案馆**是一个融合了 AI 分析、3D 可视化和诗意交互的沉浸式梦境记录平台。

### 🎯 核心功能

- 📝 **梦境记录** - 记录和保存你的梦境
- 🎭 **情感分析** - AI 分析梦境情感类型
- 🔮 **梦境解读** - 赛博周公 AI 解梦
- 🌌 **3D 星云图谱** - Three.js 程序化生成梦境星云可视化
- 📊 **统计仪表板** - 梦境情感分布统计
- 🎨 **诗意交互** - 诗意化的梦境内容

## 🚀 快速开始

### 在线体验

**Demo 预览**：[dream-nebula.vercel.app](https://dream-nebula.vercel.app)

### 本地运行

\`\`\`bash
git clone https://github.com/ach3rry/dream-nebula-archive.git
cd dream-nebula-archive
npm install
npm run dev
\`\`\`\

访问 http://localhost:3000

### Demo 模式配置

创建 \`.env.local\` 文件：

\`\`\`env
NEXT_PUBLIC_DEMO_MODE=true
\`\`\`\

## 🛠️ 技术栈

- **前端**：Next.js 15.5 + TypeScript + Tailwind CSS
- **3D 渲染**：Three.js + React Three Fiber
- **后端**：Python + FastAPI
- **数据库**：YashanDB
- **AI 引擎**：DeepSeek API
